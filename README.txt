bizbook is a two sided social network that connects professionals to clients. The network is built upon Elgg (open source social network). However, the core of Elgg (which is a flat one side network) has been modified to support two type of users 1) businesses 2) clients.

Bizbook uses facebook profile of a client to discover his/her interest profile, refered to as User Interest Fingerprint (UIF). UIF is the base to the network recomendation algorithm.
