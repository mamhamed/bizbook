bizbook is a two sided social network that connects professionals to clients. The network is built upon Elgg (open source social network). The core of Elgg (a flat one side network) has been modified to support two user types 1) businesses 2) clients.

Bizbook connects to client Facebook profile and learns his/her interest profile, refered to as User Interest Fingerprint (UIF). UIF is the base to the network recomendation algorithm.
